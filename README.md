# 👻 ObsidianGhost — Emerging Trends Guide

> "Made with ❤️, exam panic 😭, and zero preparation… until the night before."

A visually immersive and interactive **study guide web app** designed for **B.Tech (ECE) students** to quickly revise **Emerging Trends (Physics Group – II Sem)** in the most efficient way possible.

---

## 🚀 Features

✨ **Interactive UI/UX**
- Smooth landing animation
- Custom cursor (desktop)
- Minimal + aesthetic dark theme

🔍 **Smart Search System**
- Instant topic search
- Jump-to-section navigation
- Fast filtering across all units

📚 **Structured Content**
- 📋 Strategy (exam-focused tips)
- 🚀 Unit 1 (Space, Satellites, Rockets)
- 🛸 Unit 2 (UAV, Drones, Applications)
- ⚡ 2-Mark Questions (most important)

⚡ **Exam-Oriented Design**
- Priority-based topics
- Short + long answer optimization
- Ready-to-revise format

---

## 🧠 What Makes It Different?

- Focused on **maximum marks, minimum time**
